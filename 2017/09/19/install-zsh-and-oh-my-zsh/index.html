Install zsh and oh my zsh

检查是否安装zsh

cat /etc/shells

显示没有安装zsh

安装zsh包

yum -y install zsh

再次检查是否安装zsh

显示已经安装zsh

切换shell至zsh

chsh -s /bin/zsh

重启连接检查是否切换到zsh

echo $SHELL

已经切换到zsh

安装oh my zsh

官网：http://ohmyz.sh/

sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

修改oh my zsh

vi ~/.zshrc

修改ZSH_THEME字段即可，修改完成后

source ~/.zshrc
