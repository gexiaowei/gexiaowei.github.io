将非官方扩展程序加入chrome的白名单

  • chrome
  • crx
  • whitelist

在chrome中手动添加非官方扩展时，会出现重启浏览器以后无法启用扩展的情况（如图所示：）

最近群友提供了一个方法，特此弄一个中文教程；

原文地址：https://hencolle.com/2016/10/16/baidu_exporter/

注意：该方法只适用于 通过 crx 文件安装的扩展（将crx文件拖入chrome扩展程序界面）

首先下载一个描述文件：

Github地址：

https://gist.github.com/Explorare/be3dd598289252698cd37bca04abd0fe#file-com-google-chrome-mobileconfig

打开 chrome的 扩展程序 界面，复制 扩展的 ID。

用文本编辑器打开下载好的com.google.Chrome.mobileconfig找到如图所示位置：

这里的 三个 中分别对应单个扩展的ID，如果只需要一个的话，可以吧多余的两个删掉，将其中的一个标签中的值替换为第二步复制过来的 ID。编辑完后保存。如图：

双击打开第三步编辑好的描述文件，按照弹出的提示点击 继续-安装（输入电脑密码），安装结束后重启浏览器，就可以勾选这个扩展了，并且以后重启浏览器也不会有停用的提示了！
