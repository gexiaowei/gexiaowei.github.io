  • node

前言

这里会不定期的更新一些在开发工作中运用到的比较实用性的node插件

静态服务器

httpster

一个简单的http静态服务器

安装方法

npm -g install httpster

使用方法

httpster -p <port> -d <directory>

可以指定本地端口和需要访问的本地路径

port和directory为可选选项

port：默认3333

directory为控制台的当前路径

browser-sync

同步更新项目的一个静态http服务器，当项目有指定文件更改后自动刷新网页

安装方法

npm -g install browser-sync

使用方法

browser-sync start --server --files "**/*.*"

--files 需要监听的文件

本地端口默认为3000
